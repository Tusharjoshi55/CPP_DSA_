2.Access Modifiers

Private :- variables and method(functions) are accessible inside a class. It is default value of class when not mention.
Public :- variable and method are accessible to everyone in the programm.
Protected :- variable(data) and method are accessible to class and it's drived class. //inheritance topic.

3. Encapsulation
It is wrapping up of data(veriables) and method(member function) in a single unit called Class.
It is used in data hiding using access Modifiers. use private
for eg.

//normal function
void helloWorld(){
    cout<<"Hello, World !";
}
int main(){
    helloWorld();
    return 0;
}

//Encapsulation :- wrapping data and method inside a class
class Print{
    private:
    void helloWorld(){
        cout<<"Hello, World !";
    } 
}

int main(){
    Print obj;
    //now we can not use this function because it is hidden inside a wrapper or it's abstracted
    obj.helloWorld();
    return 0;
}


//Constructor

Special function invoked automatically at the time of object creation. Used for initilization
-Same name as class name
-Constructor dosen't have return type
-Only calls once(automatically), at object creation.
-Memory allocation is happen when constructor is called
-if we not defined a constructor cpp automatically create one for class

types of constructor

eg. class ClassName{

b. parameterized constructor
