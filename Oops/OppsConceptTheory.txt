3. Encapsulation
It is wrapping up of data(veriables) and method(member function) in a single unit called Class.
It is used in data hiding using access Modifiers. use private
for eg.

//normal function
void helloWorld(){
    cout<<"Hello, World !";
}
int main(){
    helloWorld();
    return 0;
}

//Encapsulation :- wrapping data and method inside a class
class Print{
    private:
    void helloWorld(){
        cout<<"Hello, World !";
    } 
}

int main(){
    Print obj;
    //now we can not use this function because it is hidden inside a wrapper or it's abstracted
    obj.helloWorld();
    return 0;
}
